## 下载

### Docker

```
docker pull apachecn0/apachecn-javaweb-zh
docker run -tid -p <port>:80 apachecn0/apachecn-javaweb-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-javaweb-zh
apachecn-javaweb-zh <port>
# 访问 http://localhost:{port} 查看文档
```
